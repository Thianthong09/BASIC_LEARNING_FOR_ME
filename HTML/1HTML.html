ກຳນົດຮູບແບບ Character encoding ໃນໜ້າເວັບ ການກຳນົດຮູບແບບການເຂົ້າລະຫັດອັກສອນ (Character encoding) ໂດຍໃຊ້ແທັກ ກຳນົດ Attribute Character ລົງໄປ ໂຄງສ້າງພາສາ HTML ໂຄງສ້າງພາສາ HTML ຈະແບ່ງອອກເປັນ 2 ສ່ວນໄດ້ແກ່ສ່ວນ head ແລະ ສ່ວນ body ໂດຍລຽງຈາກTag ແລະ ຕາມລຳດັບ ໂດຍທັງ 2 ຈະຢູ່ພາຍໃນ ສ່ວນ head ເປັນສ່ວນທີ່ຢູ່ພາຍໃນ ໃຊ້ສຳລັບອະທິບາຍຂໍ້ມູນກ່ຽວກັບເວັບເຊັ່ນ: ຊື່ເລື່ອງຂອງເວັບໄຊ ຊື່ຜູ້ສ້າງເວັບໄຊ ແລະ ເພຶ່ອໃຊ້ສຳລັບໃຫ້ຜູ້ໃຊ້ຄົ້ນຫາຂໍ້ມູນກັບເວັບໄດ້ body ເປັນສ່ວນທີ່ຢູ່ລະຫວ່າງ ໃຊ້ອະທິບາຍເນື້ອໃນຫຼັກຂອງເວັບ ເຊັ່ນ: ໃສ່ຂໍ້ຄວາມຕ່າງໆ ຮຸບພາບ, ແບບຟອມ, ວິດີໂອ, ແລະ ຍັງສາມາດກຳນົດຄຸນສົມບັດພື້ນຖານຂອງ ເວັບໄດ້ ເຊັ່ນ: ຮູບແບບພື້ນຫຼັງ ສີຂອງຕົວອັກສອນ HTML Element - ທຸກຄຳສັ່ງທີ່ຢູ່ລະຫວ່າງແທັກເປີດ ແລະ ແທັກປິດ HTML Element ບາງຢ່າງຖ້າບໍ່ມີເນື້ອໃນ (Content) ຊຶ່ງຈະຈົບຄຳສັ່ງໃນແທັກເປີດເລີຍ - ໂດຍສ່ວນໃຫຍ່ HTML Element ມັກຈະມີ Attribute ປະກອບຢູ່ໃນແທັກ Example:
